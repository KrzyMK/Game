## General Information
Project **Game** is a simple game where you can shot to enemy to win created with Unity.

## Technologies
Project is created with:
- Unity Hub 2.4.3
- Unity version 2020.3.10f1

## Features
- The user can shot to the enemy
- Must kill boss to win
